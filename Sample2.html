ORDERED LISTS CAN BE VERY SIMPLE

  2. ITEM 2
  3. ITEM 3

Ordered List Using Lowercase Letters

  1. ITEM 1
  2. ITEM 2
  3. ITEM 3

Ordered List Using Uppercase Roman Numerals

  1. ITEM 1
  2. ITEM 2
  3. ITEM 3

Ordered List Using Lowercase Roman Numerals

  1. ITEM 1
  2. ITEM 2
  3. ITEM 3

Ordered List Using Arabic Numerals (The Default)

  2. ITEM 2
  3. ITEM 3
